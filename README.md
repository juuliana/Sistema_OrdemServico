## Requisitos:
- Instalar o NetBeans ou utilizar o Eclipse;
- Instalar o MySQL;
- Instalar o WampServer ou outro servidor online que você costuma utilizar;
- Instalar o TIBCO Jaspersoft Studio ou outro para relatórios.
